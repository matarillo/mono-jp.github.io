Mono for Linux は各種ディストリビューション用のものが入手できます。

Mono環境のインストールと構成については、インストールガイドに詳しい情報があります。

Xamarin パッケージ

以下のパッケージはXamarinによってビルドされ、テストされて配布されています。Monoの安定版, 公式版, 最新版を使いたい場合はこれらをご利用ください。

  • Debian, Ubuntu, および派生ディストリビューション
  • CentOS, Fedora, および派生ディストリビューション
  • openSUSE および SLES

Linux ディストリビューションパッケージ

Monoはサードパーティーメンテナーによって、各種ディストリビューション向けにパッケージ化されています。ディストリビューションやメンテナーによっては、Monoの最新版を出荷していないパッケージもあります。

  • Arch Linux
  • CentOS EPEL
  • Debian
  • Fedora
  • Gentoo
  • openSUSE
  • Ubuntu

継続的インテグレーション(CI)パッケージ

私達のリポジトリにコミットされるたびに自動的に配布パッケージをビルドします。最先端のMonoを利用し、安定性を求めない場合はこれらをご利用ください。

CIパッケージについて詳しく知る

コミュニティパッケージ

以下のパッケージはコミュニティが提供するもので、利便性のためにここからリンクしています。これらはXamarinによってテストされたものではありませんので、自己責任でご利用ください。

  • Mono パッケージ（ Debian および Ubuntu の/opt にインストールされます）
  • Mono パッケージ（ CentOS の/opt にインストールされます）
  • Fedora Rawhide および過去のFedora向け 最新のFedora mono パッケージ
  • 最新の SUSE mono パッケージ（公式の "Factory" リポジトリに含まれます）
  • SlackPack リポジトリに含まれる最新の Slackware mono パッケージ
