Monoプロジェクトに貢献する

Mono はオープンソースプロジェクト です。ソースコードは GitHub 上で公開されています。バグフィックスや新機能の追加で貢献するには、リポジトリをフォークしてプルリクエストを送ってください！

詳しい情報、および開発ワークフローについては、コントリビューションガイドをご覧ください。

バグ報告

正しく動作しない箇所を見つけたときは、Xamarin の Bugzilla にバグレポートを登録していただければ幸いです。

レポートには、問題の詳しい説明と再現に必要な手順を含めるようにしてください。小さなテストケースを追加していただけると、私達が対応しやすくなります。

よいバグレポートを登録する方法については、バグ報告ガイドに詳しい説明があります。

困ったときは

英語のメーリングリストがいくつかありますので、Monoに関することでしたらそちらにご質問ください。メーリングリストは掲示板型インターフェースからも利用できます。

他には、IRCチャネルのチャットで質問していただくこともできます。IRCサーバーに接続するには、IRCに詳しい説明があります。

Stack Overflow はソフトウェア開発に関する有名な Q&A サイトです。Monoについてもこちらで質問をすることができます。Monoコントリビューターとコミュニティメンバーの多くが、mono タグがつけられた Stack Overflow の質問に頻繁に回答しています。質問するときは mono タグをつけるようにしてください。
